Tentang Saya

Saya adalah salah satu mahasiswa di Politeknik Negeri Madiun dari program studi D4 Teknologi Rekayasa Perangkat Lunak. Saat semester 1 perkuliahan, saya sempat merasa ragu dan bingung karena tidak memiliki latar belakang dalam pemrograman.
Tetapi, hambatan tersebut tidak memberhentikan saya untuk beradaptasi secara bertahap-tahap. Pertemuan demi pertemuan kuliah sudah saya lewati, saya mulai bisa memahami bahasa-bahasa pemrograman seperti SQL, PYTHON, HTML, JAVA, CSS, dan JAVASCRIPT. Meskipun saya anak pendiam, hal tersebut tidak akan menjadi masalah bagi saya untuk tetap menambah teman. Terima kasih telah meluangkan waktu untuk mempelajari lebih lanjut tentang saya! Saya selalu terbuka terhadap teknologi ataupun game, jadi jangan ragu untuk menghubungi saya dan menyapa!
